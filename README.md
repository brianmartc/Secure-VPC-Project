# Secure-VPC-Project
- Hands on project building a secure VPC on AWS with bastion host, private server, and controlled SSH access


## Overview
This project demonstrates a secure VPC architecture in AWS, including:
- Public subnet with public admin server
- Key based SSH access
- Controlled network acces via security groups

## How to Use / View
- Project documentation is in 'Project_Plan.md'
- Screenshots and diagrams are in the 'images/' folder

## Key Features
- Secure network isolation
- Public server acting as a bastion host
- Private server inaccesible from the internet
